# 3Dセルオートマトンの実装

## 起動方法

```
$ make 3D_LifeGame
$ ./3D_LifeGame
```
によって起動する。

初期値の操作は20~25行目で行う。
セルの更新設定(survive, birth配列)によってかなり変わるので、
変えてみると面白い。


## 操作方法

動作画面の左上に従う。

### 通常時
|キー入力|動作内容|
| :--: | :--:|
|v | 自動移動モードへの移行/通常モードへの復帰|
|s | 一時停止/再開|
|r | セルの初期化|
|f | セル更新を早める|
|d | セル更新を遅くする|
|l | カメラ位置をリセットする|
|c | セル初期化時の状態の変更|
|n | 色の変更|
|e | 編集モードへの移行|
|q | 終了|

### 編集モード時

|キー入力|動作内容|
| :--: | :--:|
|i/o |z方向の移動|
|上下左右 | x,yの移動|
|p| セルの状態の変更|
